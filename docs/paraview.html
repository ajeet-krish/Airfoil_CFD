Visualization Recipes

1. Mach Number Contours

Portfolio value: Shows compressibility effects. At M$_\infty$=0.15, peak local Mach reaches ~0.17 near the suction peak on the leading edge. Proves the flow is safely subsonic (no shocks).

  1. Open flow_results_{aoa}.vtu.
  2. In Properties: set Coloring to Mach.
  3. Click Edit Color Map → choose Rainbow Uniform or Cool to Warm.
  4. For filled contours: Filters → Alphabetical → Contour, set Contour By to Mach, set number of contours to 20.
  5. Tip: Add a Calculator filter with sqrt(Velocity_X^2+Velocity_Y^2+Velocity_Z^2) / 340.0 to compute Mach from velocity in real-time.

2. Y+ Surface Visualization

Portfolio value: Validates the boundary layer mesh quality. Y$^+$ < 1 is required for the Spalart-Allmaras turbulence model. Our mesh achieves Y$^+_{\text{max}}$ = 1.99, Y$^+_{\text{mean}}$ = 0.01 - excellent resolution.

  1. Open flow_results_{aoa}.vtu.
  2. Set Coloring to Y_Plus.
  3. Filters → Alphabetical → Extract Surface to show only wall-adjacent cells.
  4. Use Log Scale in the color map editor for better visibility of the thin high-Y$^+$ region at the leading edge.
  5. Take a screenshot zoomed into the LE region showing the structured quad BL with Y$^+$ overlay.

3. Skin Friction Coefficient (Cf)

Portfolio value: Cf magnitude and direction reveal separation points (Cf = 0), transition regions, and attached flow vs separated flow. Compare across AoA to track stall progression.

  1. Open flow_results_{aoa}.vtu.
  2. Add Calculator filter. Expression: mag(Skin_Friction_Coefficient). Name: Cf_Mag.
  3. Set Coloring to Cf_Mag.
  4. For Cf vector arrows: Filters → Alphabetical → Glyph, set Glyph Type to Arrow, Vectors to Skin_Friction_Coefficient.
  5. Plot along the airfoil surface: Filters → Alphabetical → Plot Over Line along the chord. Plot Cf_x vs x/c. The zero crossing = separation point.

4. Eddy Viscosity Ratio ($\mu_t / \mu$)

Portfolio value: Shows where the turbulence model is active. High $\mu_t$ in the wake and shear layers; near-zero in the freestream. Verifies the SA model is functioning correctly.

  1. Open flow_results_{aoa}.vtu.
  2. Calculator filter: Eddy_Viscosity / 1.8e-5. Name: MuT_Over_Mu.
  3. Set Coloring to MuT_Over_Mu.
  4. The wake behind the airfoil trailing edge should show elevated values ($\mu_t/\mu \sim 5$-25 depending on AoA).
  5. Compare across AoA: at 0° the wake is thin and symmetric; at 16° it's thick and asymmetric - dramatic visualization of stall.

5. Velocity Magnitude Contours

  1. Open flow_results_{aoa}.vtu.
  2. Add Calculator: sqrt(Velocity_X^2+Velocity_Y^2+Velocity_Z^2). Name: Velocity_Mag.
  3. Filters → Alphabetical → Contour. Set Contour By to Velocity_Mag. Set 30-50 contours.
  4. For filled contours: check Compute Normals OFF, set Contour By to Velocity_Mag.

6. Streamlines / Pathlines

  1. Select flow_results_{aoa}.vtu in Pipeline Browser.
  2. Filters → Alphabetical → Stream Tracer With Custom Source.
  3. Seed Source: Change to Line Source. Place upstream of airfoil: from (-1.5, -0.5, 0) to (-1.5, 0.5, 0).
  4. Integration Direction: FORWARD. Maximum Streamline Length: 10.
  5. Click Apply. Color streamlines by Velocity magnitude.
  6. Tip for separation visualization: Place the seed line just above the trailing edge at high AoA to capture recirculation bubbles.

7. Pressure Coefficient (Cp) Distribution

Portfolio value: Classic aerodynamic curve. Compare the suction peak growth from 0° to 16°. The $C_p$ distribution directly integrates to $C_l$.

  1. Open flow_results_{aoa}.vtu.
  2. Filters → Alphabetical → Extract Surface.
  3. Filters → Alphabetical → Plot Over Line along the airfoil boundary.
  4. In the spreadsheet/chart view, plot Pressure_Coefficient vs arc_length.
  5. Invert the Y-axis (since $-C_p$ is typically plotted upward).
  6. Export CSV from each AoA, overlay in matplotlib for the portfolio - shows suction peak rising from $-C_p \approx 0$ at 0° to $-C_p \approx 5$ at 16°.

8. Mesh Inspection

Open output/aoa_{aoa}/mesh_{aoa}.vtu for a data-free mesh view:

  1. Set Representation to Wireframe.
  2. Zoom into the leading edge to see structured BL quads transitioning to triangles.
  3. Zoom out to see the full C-grid topology (semicircle inlet, rectangular outlet).
  4. Filters → Alphabetical → Glyph with 3D arrows for velocity vectors.
  5. Filters → Alphabetical → Triangle Strips to inspect element types.

9. Velocity Profiles at x/c Stations

Portfolio value: Extracts boundary layer profiles to compute displacement thickness $\delta^*$, momentum thickness $\theta$, and shape factor $H = \delta^*/\theta$ - key metrics for separation prediction.

  1. Open flow_results_{aoa}.vtu.
  2. Filters → Alphabetical → Plot Over Line.
  3. Set endpoints to a vertical line at a chord station, e.g., at x/c = 0.5: from (0.5, -0.1, 0) to (0.5, 0.1, 0).
  4. Plot Velocity_Mag (via Calculator) vs Y coordinate.
  5. Repeat at x/c = 0.25, 0.50, 0.75, 1.0. Overlay profiles to show BL thickening downstream.

10. Wake Velocity Deficit

Portfolio value: The wake profile directly relates to drag (momentum deficit). A wider/deeper wake = higher drag. Dramatic visualization of drag growth with AoA.

  1. Plot Over Line at x/c = 1.5: from (1.5, -1.0, 0) to (1.5, 1.0, 0).
  2. Plot Velocity_Mag vs Y.
  3. Compare wake width across AoA: at 0° the wake is narrow; at 16° it's massively wide due to separation.

11. AoA Sweep Animation

Portfolio value: Animating all 5 angles shows the continuous evolution from attached flow to deep stall. Despite steady RANS limitations, the transition is visually compelling.

  1. Load all 5 VTU files: flow_results_0.vtu through flow_results_16.vtu.
  2. Apply identical visualization settings to each (e.g., Mach coloring, same color scale).
  3. Save screenshots of each at the same view angle.
  4. Use an image editor or imageio to stitch into a GIF: python -c "import imageio; imgs=[imageio.imread(f'frame_{a}.png') for a in [0,4,8,12,16]]; imageio.mimsave('aoa_sweep.gif', imgs, duration=1.0)".
  5. Frame order: 0° → 4° → 8° → 12° → 16° → 12° → 8° → 4° → 0° (smooth loop).
